# Note on AI Use

AI tools, specifically GitHub Copilot and ChatGPT, were used in a limited and supportive capacity throughout this project.

AI was primarily used for:
- Assisting with Python syntax and debugging
- Generating and refining data visualization code (Plotly and Folium)
- Improving HTML and CSS formatting for the final interactive report
- Providing suggestions for structuring code and outputs

All substantive aspects of the project were completed independently. This includes:
- Selection of research topic and framing of the argument
- Data selection and filtering using ACLED
- Construction of variables and analytical approach
- Interpretation of results and written analysis

AI outputs were reviewed, modified, and integrated manually. No analytical conclusions or arguments were generated directly by AI without critical evaluation.

The use of AI was therefore limited to technical assistance and did not replace independent thinking or original analysis.
